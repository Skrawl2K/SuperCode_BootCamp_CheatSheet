Commonly used Tags:

Areas:

  1. head

    2. In this tag properties of the HTML page are set.

  2. body

    3. In this tag is the information deposited that is to be shown in the browser. Here one can find the header, main & footer

  3. header

    4. Headers, navbars, logos, etc. are placed in the header.

  4. main

    5. The Main contains the main part of the displayed web page. Here are lists like:

    • Ordered lists (enumerated list of all 1st - 2nd - 3rd - etc.)
    • Unordered lists (only marked with a bullet point at the beginning)

  5. footer

    6. In the footer there are usually further links that lead to the imprint, the address of the company that hosts the website and the like. It is the conclusion of the displayed page of the website.
